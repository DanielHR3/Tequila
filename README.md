# Tequila Livergood — Premium Landing Page

Esta es la versión modernizada de la landing page de Tequila Livergood, rediseñada para ofrecer una experiencia de lujo, responsiva y lista para **GitHub Pages**.

## 🌐 Hosting en GitHub Pages
Este proyecto ha sido adaptado para funcionar de forma estática:
1. **Frontend:** Todo el diseño (HTML/CSS) es 100% compatible.
2. **Contacto:** La lógica de PHP ha sido sustituida en `contacto.html` por una integración con **Formspree**, lo que permite recibir mensajes sin necesidad de un servidor backend.
3. **Despliegue:** Simplemente sube los archivos a tu repositorio y activa GitHub Pages desde los Settings.

**Nota:** El archivo `contacto.php` se mantiene en el repositorio por si deseas alojarlo en un servidor que soporte PHP en el futuro.
